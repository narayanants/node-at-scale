NPM Best Practices:
--------------------

    * npm --version
    * npm version (gives npm modules version)
    * npm init --yes
    * npm help
    * npm init --yes

    nvm alias <alias_name> <version_number>

    npm config set init.author.name YOUR_NAME
    npm config set init.author.email YOUR_EMAIL

    * npms.io
    * npm home package
    * npm bugs package
    * npm repo package
    * npm install package --save
    * npm install package --save-dev
    * npm shrinkwrap - It can return the version of the current package, the Node.js version you are using and OpenSSL or V8 versions.
    * npm outdated
    * npm install --production
    * NODE_ENV=production npm install
    * .npmrc
    * .npmignore
    * .gitignore
    * npm link
